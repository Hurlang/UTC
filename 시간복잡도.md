# 시간복잡도

## 선택하기 (비서문제, 최적 정지 알고리즘)

1. 0명 살펴보기 전략

| 순번 | 1번만남 | 성공유무 |
| ---- | ------- | -------- |
| 1    | 시유    | **성공** |
| 2    | 제이손  | 실패     |
| 3    | 존      | 실패     |



2. 1명 살펴보기 전략 (*처음 만난 사람을 무조건 안만남*, 매력 : 시유>제이손>존 순서)

   이전에 만난 사람보다 좋은 사람을 바로 고른다.

   | 순번 | 1반 만남 | 2번 만남 | 3번 만남 | 성공유무 |
   | ---- | -------- | -------- | -------- | -------- |
   | 1    | 시유     | 제이손   | 존       | 실패     |
   | 2    | 시유     | 존       | 제이손   | 실패     |
   | 3    | 제이손   | 시유     |          | 성공     |
   | 4    | 제이손   | 존       | 시유     | 성공     |
   | 5    | 존       | 시유     |          | 성공     |
   | 6    | 존       | 제이손   |          | 실패     |

   

3. N명 살펴보기 전략

   - N이 커질수록 30%대로 접어든다.
   - 적어도 3개를 살펴보고 이전보다 좋은 것을 고르는 것이 만족도(성공확률)가 높다.

   | 명수 | 살펴보기 수 | 성공확률 |
   | ---- | ----------- | -------- |
   | 4    | 1           | 45%      |
   | 5    | 2           | 43%      |
   | 6    | 2           | 42%      |
   | 7    | 2           | 41%      |
   | 10   | 3           | 39%      |
   | 50   | 18          | 37%      |
   | 100  | 37          | 37%      |
   | 1000 | 369         | 36%      |

4.  위 전략이 사용될 수 있는 예

   - 자취방 구하기
   - 맛있어 보이는 음식점 찾아다니기
   - 마음에 드는 옷 고르기
